get up and running:

1. install meteor: https://www.meteor.com/install

3. git clone https://github.com/aneudy1702/platano-power-bingo.git

4. cd into the app directory `cd platano-power-bingo`

5. in the app root directory run:
	a) meteor npm install
	b) meteor

3. Open your web browser and go to http://localhost:3000 to see the app running.
